Simon Say's Game

Welcome to the Simon Says game, a memory challenge that will test your ability to remember and repeat patterns! The game presents a sequence of colors that light up, and your task is to click the same colors in the same order. Be careful, though — if you make a mistake, the game will end!
How to Play:
Start the Game : Press any key to begin the game. The game will start with a random sequence of colors.
Follow the Sequence : The game will flash the colors in a specific order. You must repeat the sequence by clicking on the colored buttons in the same order.
Keep Going : With each successful sequence, the game will increase in difficulty, adding another color to the sequence.
Game Over : If you click the wrong color or order, the game will end, and your score will be displayed. Press any key to start again.
Key Features: Levels: The game increases in difficulty with each level. Max Score: Try to beat your own high score as you progress through more levels! Good luck, and remember — focus, follow the sequence, and don’t make a mistake!
